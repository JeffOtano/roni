# iOS Auth Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add email/password auth to the iOS app using the existing `@convex-dev/auth` backend, with Keychain token persistence and automatic refresh.

**Architecture:** The iOS app calls the same `auth:signIn`/`auth:signOut` Convex actions the web uses. Tokens (JWT + refresh) are stored in Keychain and fed to `ConvexClientWithAuth` via a custom `AuthProvider`. A new `AuthManager` coordinates auth state for SwiftUI views.

**Tech Stack:** SwiftUI, ConvexMobile (convex-swift SDK), Security framework (Keychain), iOS 17+

**Spec:** `docs/superpowers/specs/2026-03-24-ios-auth-design.md`

- [ ] **Step 1: Create KeychainHelper**

```swift
import Foundation
import Security

enum KeychainHelper {
    private static let service = "coach.tonal.app"

    static func save(key: String, value: String) {
        guard let data = value.data(using: .utf8) else { return }
        let query: [String: Any] = [
            kSecClass as String: kSecClassGenericPassword,
            kSecAttrService as String: service,
            kSecAttrAccount as String: key,
        ]
        SecItemDelete(query as CFDictionary)
        var addQuery = query
        addQuery[kSecValueData as String] = data
        addQuery[kSecAttrAccessible as String] = kSecAttrAccessibleAfterFirstUnlock
        SecItemAdd(addQuery as CFDictionary, nil)
    }

    static func read(key: String) -> String? {
        let query: [String: Any] = [
            kSecClass as String: kSecClassGenericPassword,
            kSecAttrService as String: service,
            kSecAttrAccount as String: key,
            kSecReturnData as String: true,
            kSecMatchLimit as String: kSecMatchLimitOne,
        ]
        var result: AnyObject?
        let status = SecItemCopyMatching(query as CFDictionary, &result)
        guard status == errSecSuccess, let data = result as? Data else { return nil }
        return String(data: data, encoding: .utf8)
    }

    static func delete(key: String) {
        let query: [String: Any] = [
            kSecClass as String: kSecClassGenericPassword,
            kSecAttrService as String: service,
            kSecAttrAccount as String: key,
        ]
        SecItemDelete(query as CFDictionary)
    }

    static func deleteAll() {
        delete(key: Keys.jwt)
        delete(key: Keys.refreshToken)
        delete(key: Keys.email)
    }

    enum Keys {
        static let jwt = "convex_jwt"
        static let refreshToken = "convex_refresh_token"
        static let email = "convex_user_email"
    }
}
```

## Task 2: PasswordAuthProvider

**Files:**

- Create: `ios/TonalCoach/Auth/PasswordAuthProvider.swift`

**Context:** Read `ConvexClientWithAuth` and `AuthProvider` protocol in the convex-swift SDK source at:
`~/Library/Developer/Xcode/DerivedData/TonalCoach-*/SourcePackages/checkouts/convex-swift/Sources/ConvexMobile/ConvexMobile.swift`

The `AuthProvider` protocol requires:

- `login(onIdToken:)` - provide JWT token, return auth data
- `loginFromCache(onIdToken:)` - restore session from stored credentials
- `logout()` - clear credentials
- `extractIdToken(from:)` - extract JWT from auth result

- [ ] **Step 1: Create PasswordAuthProvider**

```swift
import ConvexMobile
import Foundation

struct AuthData {
    let refreshToken: String
}

enum AuthError: Error, LocalizedError {
    case noStoredCredentials
    case refreshFailed(String)
    case signInFailed(String)

    var errorDescription: String? {
        switch self {
        case .noStoredCredentials: "No stored credentials"
        case .refreshFailed(let msg): "Token refresh failed: \(msg)"
        case .signInFailed(let msg): "Sign in failed: \(msg)"
        }
    }
}

final class PasswordAuthProvider: AuthProvider {
    typealias T = AuthData

    /// Reference to the ConvexClient for making auth:signIn action calls.
    /// Set before first use (ConvexClientWithAuth passes itself as the client).
    var client: ConvexClient?

    func login(onIdToken: @Sendable @escaping (String?) -> Void) async throws -> AuthData {
        // Read stored tokens from Keychain (placed there by AuthManager.signIn)
        guard let jwt = KeychainHelper.read(key: KeychainHelper.Keys.jwt) else {
            throw AuthError.noStoredCredentials
        }
        onIdToken(jwt)
        let refreshToken = KeychainHelper.read(key: KeychainHelper.Keys.refreshToken) ?? ""
        return AuthData(refreshToken: refreshToken)
    }

    func loginFromCache(onIdToken: @Sendable @escaping (String?) -> Void) async throws -> AuthData {
        // Try refresh if we have a refresh token
        if let refreshToken = KeychainHelper.read(key: KeychainHelper.Keys.refreshToken),
           let client {
            do {
                let result: SignInResult = try await client.action(
                    "auth:signIn",
                    with: ["refreshToken": refreshToken]
                )
                if let tokens = result.tokens {
                    KeychainHelper.save(key: KeychainHelper.Keys.jwt, value: tokens.token)
                    KeychainHelper.save(key: KeychainHelper.Keys.refreshToken, value: tokens.refreshToken)
                    onIdToken(tokens.token)
                    return AuthData(refreshToken: tokens.refreshToken)
                }
            } catch {
                // Refresh failed - clear stale tokens
                KeychainHelper.deleteAll()
                throw AuthError.refreshFailed(error.localizedDescription)
            }
        }

        // Fall back to stored JWT
        guard let jwt = KeychainHelper.read(key: KeychainHelper.Keys.jwt) else {
            throw AuthError.noStoredCredentials
        }
        onIdToken(jwt)
        let refreshToken = KeychainHelper.read(key: KeychainHelper.Keys.refreshToken) ?? ""
        return AuthData(refreshToken: refreshToken)
    }

    func logout() async throws {
        KeychainHelper.deleteAll()
    }

    func extractIdToken(from authResult: AuthData) -> String {
        KeychainHelper.read(key: KeychainHelper.Keys.jwt) ?? ""
    }
}

// MARK: - Response Types

/// Matches the shape returned by the auth:signIn Convex action.
struct SignInResult: Decodable {
    let tokens: TokenPair?
    let signingIn: Bool?
}

struct TokenPair: Decodable {
    let token: String
    let refreshToken: String
}
```

## Task 4: AuthManager

**Files:**

- Create: `ios/TonalCoach/Auth/AuthManager.swift`

- [ ] **Step 1: Create AuthManager**

`@Observable` class that coordinates auth operations:

- `signIn(email:password:)` - calls `auth:signIn` action with `flow: "signIn"`, stores tokens in Keychain, triggers `ConvexClientWithAuth.login()`
- `signUp(email:password:)` - calls `auth:signIn` action with `flow: "signUp"`, same token flow
- `signOut()` - calls `auth:signOut` action, clears Keychain, triggers `ConvexClientWithAuth.logout()`
- `requestPasswordReset(email:)` - calls `auth:signIn` with `flow: "reset"`
- `confirmPasswordReset(email:code:newPassword:)` - calls `auth:signIn` with `flow: "reset-verification"`, stores tokens
- `restoreSession()` - called on app launch, triggers `ConvexClientWithAuth.loginFromCache()`

Properties: `isAuthenticated`, `isLoading`, `currentEmail`, `error`

Inject via SwiftUI environment.
